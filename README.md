# Homebrew Tap for Autom8y

This repository contains Homebrew formulae for tools from the [autom8y](https://github.com/autom8y) organization.

## Installation

```bash
brew tap autom8y/tap
brew install ari
```

Or in a single command:

```bash
brew install autom8y/tap/ari
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `ari` | Ariadne CLI - The thread through the Knossos labyrinth |

## Updating

```bash
brew update
brew upgrade ari
```

## Uninstalling

```bash
brew uninstall ari
brew untap autom8y/tap
```

## Issues

For issues with the formulae, please file an issue in this repository.

For issues with the tools themselves, please file issues in their respective repositories:
- [knossos](https://github.com/autom8y/knossos) - For `ari` CLI issues
